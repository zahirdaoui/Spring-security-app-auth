# Spring MVC 5 + Thymeleaf + Spring Security (Basic Example)

This is a simple Spring MVC 5 web application that demonstrates:

- Spring MVC (Java Config)
- Thymeleaf as the template engine
- Spring Security (basic authentication)
- A Home controller that returns an HTML page

The project is meant as a small learning/demo app to understand how Spring MVC works with Thymeleaf and Spring Security.

## Tech Stack
- Java
- Spring MVC 5
- Thymeleaf
- Spring Security
- Maven

## Features
- Simple home page (`/home`)
- Basic login with Spring Security
- Thymeleaf templates for UI

   mvn clean install
